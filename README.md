# USDC Donate Button

A fun demo project that implements a stylish **donate button** integrated with a Solidity smart contract, created as part the Cyfrin courses on smart contract development and frontend integration.

This is **not** a production-ready donation system.  
This is an educational demo to explore **usability & interaction design** for engaging web buttons/animations.

## Tech Stack

- Frontend: React + Typescript + Vite and a bit of SCSS
- Blockchain: Foundry, Solidity (local anvil)
- Web3 integration: viem
- Testing: Playwright
- The visual style and animation are heavily inspired by the creative "Coin Flip Donate Button" by Cooper Goeke (https://codepen.io/coopergoeke/pen/abZqEbK)
